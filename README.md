# 实习工资计算器

## 功能说明
这个程序计算不同工作天数对月收入的影响，包含周末不休息的情况。

## 工资结构
- 日薪 **lalala**
- 每 **tratratra** 天获得一次奖金
- 奖金 **bababa**

## 文件说明

### 1. simple_test.py
简单版本，直接输出关键数据
```bash
python simple_test.py
```

### 2. run_analysis.py
包含详细分析和图表生成
```bash
python run_analysis.py
```

### 3. calc.py
从零开始写的完整原始修改版本


## 图表说明
程序会生成4个子图：
1. 工作天数 vs 总收入
2. 工作天数 vs 日平均收入  
3. 工作天数 vs 奖金获得次数
4. 工作天数 vs 总奖金

图表保存为 `salary_analysis.png`
