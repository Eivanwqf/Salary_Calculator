# 实习工资计算器

## 功能说明
这个程序计算不同工作天数对月收入的影响，包含周末不休息的情况。

## 工资结构
- 日薪：120元
- 每10天获得一次奖金：1000元
- 奖金发放日：第10、20、30天

## 文件说明

### 1. simple_test.py
简单版本，直接输出关键数据
```bash
python simple_test.py
```

### 2. run_analysis.py
完整版本，包含详细分析和图表生成
```bash
python run_analysis.py
```

### 3. calc.py
原始修改版本

## 分析结果

### 不同工作天数的收入对比
| 工作天数 | 总收入(元) | 日均收入(元) | 奖金次数 |
|---------|-----------|-------------|---------|
| 1天     | 120       | 120.00      | 0       |
| 5天     | 600       | 120.00      | 0       |
| 10天    | 2200      | 220.00      | 1       |
| 15天    | 2800      | 186.67      | 1       |
| 20天    | 4400      | 220.00      | 2       |
| 25天    | 5000      | 200.00      | 2       |
| 30天    | 6600      | 220.00      | 3       |

### 关键发现
1. **收入差距**：1天(120元) vs 30天(6600元)，差距6480元
2. **奖金效率**：每10天的奖金日使日均收入显著提升
3. **最佳工作周期**：10天、20天、30天的日均收入最高(220元)

3. 工作天数 vs 奖金获得次数
4. 工作天数 vs 总奖金
